\subsubsection*{template$<$constraint\+::decayed\+\_\+type Type, constraint\+::on\+\_\+subscribe\+\_\+fn$<$ Type $>$ On\+Subscribe\+Fn$>$\newline
class rpp\+::specific\+\_\+observable$<$ Type, On\+Subscribe\+Fn $>$}
observable specified with specific type of On\+Subscribe\+Fn. Used to store On\+Subscribe\+Fn function as is on stack (instead of allocating it somewhere). 

It has better performance comparing to \mbox{\hyperlink{classrpp_1_1dynamic__observable}{rpp\+::dynamic\+\_\+observable}}. Use it if possible. But it has worse usability due to On\+Subscribe\+Fn template parameter.
